//▰ #2.합성동사 : <10000010차수>
//▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀
//※"합성동사"란 2개의 동사가 합쳐진 형태의 동사. (ex. 開け放つ = 開ける + 放つ)

//※입력 예 (반드시 아래의 입력 형식을 따라야 합니다)

//A.뒤의 동사가 1단(=2류) 동사인 경우
//-揉み立てる 라는 동사를 입력한다면 : 揉み立て	〚주무른다〛揉み立て=る1	10000010	1
//-待ち侘びる 라는 동사를 입력한다면 : 待ち侘び	〚기다린다〛待ち侘び=る1	10000010	1

//B.뒤의 동사가 5단(=1류) 동사인 경우
//-る형 当て擦る 동사 입력 : 当て擦(?=[らりるれろっ])	〚빈정거린다〛=る5	10000010	1
//-う형 擦れ違う 동사 입력 : 擦れ違(?=[わいうえおっ])	〚엇갈린다〛=う	10000010	1
//-く형 啜り泣く 동사 입력 : 啜り泣(?=[かきくけこい])	〚흐느껴 울다〛=く	10000010	1
//-ぐ형 立ち騒ぐ 동사 입력 : 立ち騒(?=[がぎぐげごい])	〚떠들어댄다〛=ぐ	10000010	1
//-す형 踏み外す 동사 입력 : 踏み外(?=[さしすせそ])	〚이탈한다〛=す	10000010	1
//-つ형 開け放つ 동사 입력 : 開け放(?=[たちつてとっ])	〚활짝 개방하다〛=つ	10000010	1
//-ぶ형 弾け飛ぶ 동사 입력 : 弾け飛(?=[ばびぶべぼん])	〚튀어날다〛=ぶ	10000010	1
//-む형 放り込む 동사 입력 : 放り込(?=[まみむめもん])	〚던져넣는다〛=む	10000010	1

//※る형 동사의 경우 1단은 "=る1", 5단은 "=る5"를 붙여야 합니다.

//※(중요!!!!) 번역어의 어미는 ""ㄴ받침 혹은 ㄹ받침""만 지원합니다.
//예 : 빈정거리다(x), 빈정거린다(o)  //  던져넣다(x), 던져넣는다(o)

//※동사 생성 정규식을 입력 후, 어미변화 테스트를 해볼 필요가 있습니다.
//예를들어 席替える 라는 동사를 입력했다면, 席替えて(연결형), 席替えない(부정형), 席替えるつもり(예정형),
//         席替えます。(경어형),  席替えなさい。(지시형), 席替え。(명사형) 등의 활용이 잘 되는지 체크해보고
//         만약 올바른 활용이 되지 않는다면, 번역어를 달리 바꿔보는 작업이 필요합니다.

//※명사형의 사용자직접 등록을 지원합니다. (명사형 등록이 없다면 자동 처리됨)
//"揉み立てる" 라는 동사를 등록하고, 이 동사의 명사형을 "주무르기"로 하고 싶다면 :
//揉み立て	〚주무른다_주무르기〛揉み立て=る1	10000010	1
//-----------------------------------------------------------------------------
